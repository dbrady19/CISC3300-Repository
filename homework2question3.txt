3. Flexbox is used when you need simple linear layouts, it it used for things like navigation bars and responsive containers that grow or shink based on the space. CSS grid is used when you need a more robust structured layout. Am example would be designing a webpage with well defined rows and columns that align things with precision. Flexbox excels at managing the layout of items along a single direction (row/column), while CSS Grid is powerful for controlling two-dimensional layouts where both rows and columns matter.
